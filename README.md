# Personal Portfolio Website


## Structure

portfolio/

- file/
  - resume.pdf

- images/
  - profile-photo.png

- index.html

- style.css

- script.js



## How To Run


1. Download the project

2. Open index.html in browser


## Customize


Replace:

images/profile-photo.png

with your own photo.


Replace:

file/resume.pdf

with your own resume.


Edit your:

Name
Skills
Projects
Contact details

inside index.html.
